# Asteroids Clone 🚀

A classic **Asteroids-style arcade game** built with **HTML5 Canvas** and **Vanilla JavaScript**.
Fly through space, destroy asteroids, avoid collisions, and survive as long as possible.

---

## 🎮 Features

* HTML5 Canvas rendering
* Smooth ship movement & rotation
* Asteroid spawning and splitting
* Bullet shooting mechanics
* Collision detection
* Score tracking
* Keyboard controls
* Retro arcade gameplay

## 🕹 Controls

| Key       | Action         |
| --------- | -------------- |
| `W`     | Thrust Forward |
| `A`     | Rotate Left    |
| `D`     | Rotate Right   |
| `S`     | Thrust Back    |
| `Space` | Shoot          |
| `Esc`   | Pause Game     |

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/asteroids-clone.git
```

### 2. Open the project

```bash
cd asteroids-clone
```

### 3. Run locally

You can simply open `index.html` in your browser.

Or use a local development server:

```bash
# VS Code Live Server
Right Click -> Open with Live Server
```
